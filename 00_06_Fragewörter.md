## German Question Words (Fragewörter)

| German | English | Usage | Example Question | Example Answer |
|--------|---------|-------|------------------|----------------|
| **Basic Questions** | | | | |
| wer | who | person (subject) | Wer ist das? | Das ist Maria. |
| wen | who/whom | person (accusative object) | Wen siehst du? | Ich sehe Maria. |
| wem | to whom | person (dative object) | Wem hilfst du? | Ich helfe Maria. |
| wessen | whose | possession | Wessen Auto ist das? | Das ist Marias Auto. |
| was | what | thing/object | Was machst du? | Ich lese ein Buch. |
| **Location & Direction** | | | | |
| wo | where | location (static) | Wo wohnst du? | Ich wohne in Berlin. |
| wohin | where to | direction (motion to) | Wohin gehst du? | Ich gehe nach Hause. |
| woher | where from | origin (motion from) | Woher kommst du? | Ich komme aus Deutschland. |
| **Time** | | | | |
| wann | when | point in time | Wann kommst du? | Ich komme um 8 Uhr. |
| wie lange | how long | duration | Wie lange bleibst du? | Ich bleibe zwei Stunden. |
| seit wann | since when | starting point | Seit wann wohnst du hier? | Seit 2020. |
| bis wann | until when | end point | Bis wann arbeitest du? | Bis 17 Uhr. |
| **Manner & Description** | | | | |
| wie | how | manner/description | Wie geht es dir? | Gut, danke. |
| wie viel | how much | uncountable quantity | Wie viel kostet das? | Das kostet 10 Euro. |
| wie viele | how many | countable quantity | Wie viele Bücher hast du? | Ich habe fünf Bücher. |
| wie oft | how often | frequency | Wie oft gehst du ins Kino? | Einmal pro Woche. |
| **Reason & Purpose** | | | | |
| warum | why | reason | Warum lernst du Deutsch? | Weil ich in Deutschland arbeite. |
| weshalb | why | reason (formal) | Weshalb bist du hier? | Ich bin hier wegen der Arbeit. |
| wieso | why/how come | reason (informal) | Wieso kommst du so spät? | Der Zug hatte Verspätung. |
| wozu | what for | purpose | Wozu brauchst du das? | Ich brauche es für die Schule. |
| **Choice & Selection** | | | | |
| welcher/welche/welches | which | specific choice | Welches Auto gefällt dir? | Das rote Auto. |
| was für ein/eine | what kind of | type/category | Was für ein Auto fährst du? | Einen BMW. |
| **Compound Questions** | | | | |
| womit | with what | instrument/means | Womit fährst du? | Mit dem Bus. |
| wovon | about what/of what | topic/source | Wovon sprichst du? | Von der Arbeit. |
| worüber | about what | topic | Worüber denkst du nach? | Über das Problem. |
| wofür | for what | purpose/benefit | Wofür ist das gut? | Für die Gesundheit. |
| worauf | on what/for what | expectation | Worauf wartest du? | Auf den Bus. |
